# Primer Contacto con Cursor AI

**Objetivo de este documento:** Documentar mi experiencia utilizando Cursor como IDE asistido por IA en el proyecto PulseTasks.

## Interfaz explorada
- Explorador de archivos con vista previa de IA
- Terminal integrado
- Chat lateral (Ctrl+K)
- Composer (Ctrl+L) — el más poderoso

## Atajos que más uso
- Ctrl + K → Chat contextual
- Ctrl + L → Compositor (cambia varios archivos al mismo tiempo)
- Ctrl + I → Edición en línea
- Alt + L → Aceptar sugerencia

## Ejemplos reales donde Cursor mejoró mi código
1. Refactorizó `createTaskElement()` en 8 segundos (mejoró la legibilidad y el modo oscuro).
2. Añadió validación de formulario + mensaje de error con animación.

**Conclusión:** Cursor es 3 veces más rápido que VS Code + Copilot. Lo recomiendo encarecidamente.
