# PulseTasks 

Gestor de tareas moderno y completo - Full Stack

Una aplicación full-stack para gestionar tareas con diseño moderno, modo oscuro, múltiples idiomas y drag & drop.

---

 Funcionalidades:

 - Crear, editar y eliminar tareas
 - Marcar como completadas
 - Drag & Drop para reordenar tareas
 - Filtros por prioridad, categoría y estado
 - Estadísticas en tiempo real
 - Modo oscuro / claro
 - Soporte para 3 idiomas (Español, Portugués, Inglés)
 - Botón para eliminar todas las tareas completadas
 - Botón para marcar todas como completadas
 - API RESTful con Express
 - Backend desplegado (Render)

---

 Tecnologías utilizadas

**Frontend:**

HTML5, CSS3, Tailwind CSS v4
JavaScript vanilla (ES6+)
SortableJS (Drag & Drop)

**Backend:**

Node.js + Express
MongoDB (Atlas) – próximamente
CORS y Swagger (documentación)

**Deploy:**

Backend: Render.com
Frontend: Vercel (próximamente)

---

Cómo ejecutarlo localmente

Requisitos previos:

Node.js instalado

Backend:

```bash
cd server
npm install
npm run dev 
```
